About Me

I am a photographer hobbiest and an aspiring software engineer based out of Portland, OR. I live a rather minimalistic lifestyle and I enjoy carrying over these principles into the code that I write and love to incorporate this theme into projects that I take part in. I am a firm believer in "less is more". Recently have graduated from Launchcode's lc101 program, I have been spending my days exploring front-end design and user experience with javaScript and Jquery.

Projects

ParPal

This web application is a tool for golfers to be able to keep track of their scores and the scores of their opponents on the fly. My contributions included:

  • Create the model classes for the relational database
  • Route Handling within the controllers
  • Implementing application logic within the controller
  • Implementation of signup/login functionality as well User input validation U/I testing

North West Mothers Milk Bank

We built a mobile friendly web application allows a user to sign up and track their milk donations. Within the app a user is able to login and logout, find donation sites in the Portland area and input when the donation was given and at what location. There is a map which displays donation sites around the city, and a table which displays all of the donation records.
